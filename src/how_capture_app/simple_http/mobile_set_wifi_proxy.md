## iOS中设置Wifi代理为Charles

以`iPhone 6`为例来解释，如何给`iOS`设置Wifi代理为Charles

`设置 -> Wifi -> 点击你的Wifi -> HTTP代理 -> 配置代理`

* 手动
  * 默认是关闭，此处改为手动
* 服务器：`10.108.129.57`
* 端口：`5678`
